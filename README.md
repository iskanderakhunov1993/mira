# FitFlow

Нативный SwiftUI-прототип фитнес-приложения: дневник питания по фото, вода,
персональные тренировки и отслеживание прогресса.

## Запуск

1. Установить Xcode 16 или новее.
2. Открыть `FitFlow.xcodeproj`.
3. Выбрать команду разработки в Signing & Capabilities.
4. Запустить target `FitFlow` на iPhone Simulator с iOS 17+.

Сейчас AI и покупка подписки работают в демонстрационном режиме. Архитектура
оставляет отдельный интерфейс `AIAnalyzing` для подключения серверного API.

Подробное продуктовое решение находится в [docs/MVP.md](docs/MVP.md).
